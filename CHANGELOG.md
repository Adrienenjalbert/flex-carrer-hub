## [1.0.0] - 2025-01-07

### Added

#### Interactive Tools (9)
- **Paycheck Calculator** — Take-home pay estimates with 2025 tax data for all 50 states
- **Tax Estimator** — 1099/W-2 quarterly tax planning with federal brackets
- **Shift Income Planner** — Weekly/monthly earnings projection calculator
- **Childcare Calculator** — Break-even analysis with state-specific childcare costs
- **Commute Calculator** — True commuting cost including time value of money
- **Unemployment Calculator** — State-by-state benefit estimates
- **Career Path Explorer** — Visual role progression mapping for 20+ careers
- **Skills Analyzer** — Gap analysis with personalized recommendations
- **WorkTalk** — Bilingual (English/Spanish) workplace phrases with text-to-speech

#### Programmatic SEO Pages
- 19 Indeed Flex location pages (active US metro markets)
- 20+ role pages with detailed job information
- 4 industry pages (Hospitality, Industrial, Retail, Facilities)
- Location × Role combination pages for hyper-local SEO
- Financial tips and educational guides

#### SEO Infrastructure
- Schema.org structured data (JobPosting, FAQPage, HowTo, Article, BreadcrumbList)
- Dynamic XML sitemap with 200+ URLs
- Unique meta tags and Open Graph for every page
- LLM-friendly `/llm.txt` endpoint
- Optimized internal linking structure

#### Data Files
- 50-state income tax rates (2025)
- 50-state childcare cost data (2024)
- 50-state unemployment benefit rules (2024)
- 19 Indeed Flex market locations
- 20+ job role definitions
- 250+ workplace English phrases for Spanish speakers

#### UI/UX
- Mobile-first responsive design
- shadcn/ui component library integration
- Dark/light mode support
- Accessible ARIA-compliant components
- Fast page loads with Vite optimization

### Technical
- React 18 with TypeScript
- Vite 5 build system
- Tailwind CSS with semantic design tokens
- TanStack Query for data fetching
- React Router v6 for navigation
- Supabase backend via Lovable Cloud
